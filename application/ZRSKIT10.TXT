        How I am writing the Documentation.
        -----------------------------------
            Every utility listed here has a "README" file which
        is a short one page explaintation on how to use that program.
        Every application also has a "DOC" file, which is has the same
        filename as the utility and a file name extension of ".DOC".
        Such as PLAY.EXE has a document file named PLAY.DOC.


        PLAY.EXE        - Full screen sound player and editor


        SPLAYER.EXE     - command line sound player, throught the PCs
                          internal speaker

        SPLAYIT.EXE     - point and shoot sound player, which can use
                          the internal PC speaker or the printer port
                          for sound output (source code available)

        SCONVERT.EXE    - sound converion program to convert sound files
                          from signed, like sounds from the Amiga, to
                          unsigned format

        SREVERB.EXE     - command line program to add reverb or echo to
                          digital sound samples

        ORDER.FRM       - ORDER FORM - so you can order one of our
                          AUDIO BYTES to improve you sound playback

        ZRSKIT10.DOC    - THIS file.


        SPLAYIT.PAS     - TURBO PASCAL source code for SPLAYIT.EXE


        BUGS, FEATURES  - The new bugs and newest features of of PLAY.EXE


        readme (intro)  - A short introduction for BBS sysops and those
                          of you that can't wait to get started.
